# 🚀 Mex Toto On Live Future

**Desarrollo de sistema wearable de proyección digital y portal holográfico.**

Este proyecto representa la evolución digital del entretenimiento en vivo, integrando tecnología vestible y realidad aumentada para crear una experiencia inmersiva única.

## 🌟 Visión del Proyecto
Transformar el concepto de espectáculo tradicional mediante el uso de:
* **Wearable Projection:** Un sistema de proyección digital integrado en la vestimenta.
* **Portal Holográfico:** Interacción con elementos digitales y códigos escaneables (Códice Totò).
* **Innovación Creativa:** Fusión de electrónica DIY, reciclaje creativo y tecnología de vanguardia.

## 🛠️ Componentes Técnicos
* **Hardware:** Módulos Bluetooth, motores, LEDs y sistemas de proyección compactos.
* **Software:** Integración de realidad virtual (VR), videos 360° y contenido digital interactivo.
* **Concepto:** Digitalización de la identidad escénica mediante herramientas de código abierto.

## 📁 Estructura del Repositorio
* `/docs`: Documentación conceptual y diagramas.
* `/hardware`: Esquemas de circuitos y componentes electrónicos.
* `/assets`: Recursos visuales y logotipos.
